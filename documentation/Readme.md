# Project Description

This is a donation website built using HTML, CSS and Javascript.You can donate to support the poor.


Hi! I'm **Unnati Makrariya**, I have created this donation website using HTML, Javascript and CSS.


**For Frontend** - ` npm i `

## Features

- Home Page
- Blog Page
- Causes Page
- Event Page
- Blog Page
- Pages Page
- Contact Page
